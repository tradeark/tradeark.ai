# GET /market/ticker

返回一个归一化后的单标的 ticker 快照。

## 查询参数

| 参数 | 必填 | 说明 |
| --- | --- | --- |
| `exchange` | 是 | 交易所标识 |
| `symbol` | 是 | 要查询的交易对 |

## 响应字段

| 字段 | 说明 |
| --- | --- |
| `symbol` | 归一化后的交易对 |
| `last` | 最新成交价 |
| `change_24h` | 24 小时涨跌幅 |
| `high_24h` | 24 小时最高价 |
| `low_24h` | 24 小时最低价 |
| `volume_24h` | 24 小时成交量 |
