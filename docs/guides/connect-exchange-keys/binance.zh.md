# Binance

这一页演示怎么为 TradeArk 创建 Binance API 密钥。

如果你还没有 Binance 账户，可以先通过这里注册：

[Binance 注册链接](https://www.binance.com/join?ref=TradeArk)

Binance API 管理地址：

`https://www.binance.com/en/my/settings/api-management`

## 创建密钥

1. 登录 Binance 并开始 API 创建流程。

![Binance API 创建入口](../../assets/exchange-api/binance-create-start.png)

2. 选择系统自动生成的方式，然后继续并填写一个 API 备注名称。

![Binance 系统生成选项](../../assets/exchange-api/binance-system-generate.png)

3. 按要求填写验证码、邮箱验证或 2FA，完成创建。

![Binance 验证并创建](../../assets/exchange-api/binance-verify-create.png)

4. 按图中方式设置 API 权限，只保留交易相关权限并保存。

Binance 展示出 API Key 和 Secret Key 后，继续看 [接入 TradeArk](TradeArk.md)。
