# Bybit

这一页演示怎么为 TradeArk 创建 Bybit API 密钥。

如果你还没有 Bybit 账户，可以先通过这里注册：

[Bybit 注册链接](https://www.bybit.com/invite?ref=4LORQ0)

Bybit API 管理地址：

`https://www.bybit.com/app/user/api-management`

## 创建密钥

1. 登录 Bybit 并打开 API 管理页面。
2. 点击右上角的创建按钮，选择系统生成 API Key 的流程。

![Bybit 创建 API 按钮](../../assets/exchange-api/bybit-create-button.png)

3. 按图中方式勾选需要的权限，完成 API 创建。

![Bybit 权限设置](../../assets/exchange-api/bybit-permissions.png)

4. 按图中方式保存权限设置，然后把生成的 API Key 和 Secret Key 留给下一步接入使用。

创建完成后，继续看 [接入 TradeArk](TradeArk.md)。
